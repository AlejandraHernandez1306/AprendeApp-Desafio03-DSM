# AprendeApp - Aplicación de Recursos de Aprendizaje

Aplicación para Android desarrollada en Kotlin que permite a los estudiantes registrarse, iniciar sesión y acceder a una lista de recursos educativos, consumiendo datos desde una API REST creada en MockAPI.io.

## Funcionalidades Implementadas

*   **Autenticación Completa:** Registro de usuario con validación de contraseña segura, Login y Logout con gestión de sesión persistente.
*   **Gestión de Recursos (CRUD):**
    *   **Crear:** Agregar nuevos recursos a través de un formulario.
    *   **Leer:** Visualizar la lista completa de recursos.
    *   **Actualizar:** Editar la información de recursos existentes.
    *   **Eliminar:** Borrar recursos con un diálogo de confirmación.
*   **Búsqueda y Filtro:** Búsqueda avanzada por título y ID, y filtro dinámico por tipo de recurso.
*   **Ordenamiento:** Opción para ordenar la lista de recursos alfabéticamente (A-Z y Z-A).
*   **Interfaz de Usuario:** Interfaz limpia basada en Material Design con feedback visual (loaders, toasts, diálogos) y navegación intuitiva.

## Tecnologías y Librerías

*   **Lenguaje:** Kotlin
*   **Arquitectura:** Patrón Repositorio
*   **Networking:** Retrofit & OkHttp
*   **UI:** Vistas XML, RecyclerView, CardView, Material Components
*   **Asincronía:** Corrutinas de Kotlin
*   **Carga de Imágenes:** Glide
